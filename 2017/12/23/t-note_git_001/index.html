平时我们

  • 工作区(workspace):平时编写代码的地方,本地的文件
  • 暂存区(stash):我们在提交到本地仓库前的一个暂存处
  • 本地仓库(repo)：提交后的本地仓库
  • 远程仓库(remote)：远端的一个代码托管的仓库

状态

  • 未跟踪(untracked)
  • 未修改(ummodified)
  • 已修改(modified)
